# Images Directory

This folder is reserved for any future image assets for the NexaTech Solutions website.

Currently, the website uses CSS gradients and Unicode emoji as visual elements,
so no external image files are required to run the website.

## Future Images
If you add images, use descriptive filenames:
- `hero-bg.webp`
- `team-ceo.webp`
- `office-sf.webp`
- etc.
